How to manage tasks
Windows Task Scheduler GUI
Open Task Scheduler from the Start menu, or run taskschd.msc in Run (Win+R). All tasks above are listed under Task Scheduler Library.
View history: right-click task → History tab Run manually: right-click → Run Edit: right-click → Properties Delete: right-click → Delete
From PowerShell
Quick commands to manage tasks from the terminal.
Get-ScheduledTask | Where TaskPath -eq "\"    # list all root tasks
Start-ScheduledTask "AIVisibility_WeeklyRefresh"    # run immediately
Disable-ScheduledTask "AIVisibility_WeeklyRefresh"    # pause
Unregister-ScheduledTask "AIVisibility_WeeklyRefresh"    # delete
